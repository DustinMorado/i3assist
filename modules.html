i3assist¶

This page contains the i3assist API and a list of module’s classes

i3assist.Euler([phi, theta, psi, unit]) Describes a particle’s orientation using ZXZ extrinsic euler angles.
i3assist.RotationMatrix([angles, unit]) Describes a particle’s orientation using ZXZ passive rotation matrix.
i3assist.GridSearch([theta_max, theta_step, ...]) Describes the local grid search implemented in new I3.
i3assist.Position(position_line) A single particle transform in old I3.
i3assist.PositionList([filename, positions]) Describes a full position file as a list of Positions.
i3assist.Transform(transform_line) A single particle transform in new I3.
i3assist.TransformList([filename, transforms]) Describes a full transform file with as a list of Transforms.
